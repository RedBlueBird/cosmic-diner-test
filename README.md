# Cosmic Diner 

Link: https://redbluebird.github.io/cosmic-diner-test/

<br/>

**Genre:** Cooking Simulation / Knowledge Roguelike / Eldritch Horror

**Summary:** Manage a diner that transitions from serving mundane humans to eldritch beings over a 30-day cycle.

<br/>

**Description:** 

A "Knowledge Roguelike" where the player must learn the chemical and metaphysical properties of food to survive.

Currently, a 5-day human arc and a boss to test the feasibility and juiciness of this idea.
